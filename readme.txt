== Description ==

Do you like time-managemente techniques like "The Pomodoro Technique"?
You'll love this plugin!!!! It allows you to set timers for both work and break and to use them when you write your posts, it is very useful in case of long posts that require hour of typing.
Just start the timer in the post editing page and write! When it's time to take a break, the plugins overlays the entire editing page and let you take a break.

== Installation ==

1. Upload `TomatoPress` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go in the admin page and set the timers duration
